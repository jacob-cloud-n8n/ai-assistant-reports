💻 全能助理 (Desktop / Code Agent)

以 Codex / ag 為例

專為複雜邏輯、系統整合及大數據操作設計，具備強大的工具調用能力。
  • ✓ 自動版本控制：串接 GitHub API 執行拉取、Rebase Pull 及金鑰自動過濾，保障資安安全。
  • ✓ 複雜工具調用：自動雙向讀寫 Google Sheets 與 Notion 預約看板，聯動 Firebase 管理。
  • ✓ 錯誤自癒子流程：n8n 執行死鎖或當機時，呼叫 error-handler 重試並主動發送 Discord 警報。
