為什麼要學 AI 素養？

多數人只會問「幫我寫...」，學完差別巨大

之前之後
幫我寫一封 email你是行銷專家，請用專業但親切的語氣，幫我寫一封回覆客戶詢價的 email...
這個怎麼做？我正在 [情境]，遇到 [問題]，你覺得最簡單的做法是什麼？

提示詞公式

角色 + 任務 + 格式 + 限制

💡 範例：
「你是資深行銷專家（角色），幫我寫一封活動邀請 email（任務），用 150 字以內、分三段（格式），語氣要專業但親切（限制）」

AI 協作黃金自律禮儀

AI 不是隨問隨答的玩具，而是有工作規範的虛擬員工

🌅 1. 開工對齊 (Start Work)

每次對話開始時無條件輸入「開工」（或執行開工腳本）。讓 AI 自動對齊目前的狀態與歷史，防止 AI 遺忘進度，避免你每次都重新解釋。

🌃 2. 收工備份 (Finish Work)

對話結束前，主動執行「收工」流程。請 AI 總結今日成果寫入記憶檔案、主動排除金鑰等敏感資料（防洩漏），並安全關閉對話。

單一真相來源 (SSOT)

AI 的對話框就像沙灘上的腳印，隨時會被潮水沖走

📌 Single Source of Truth (SSOT)：
AI 助理有失憶症（上下文長度限制）。重要的點子、程式碼或決策，**絕對不要只留在對話框裡**！必須立刻整理到你的「實體學習小卡」或個人的「第二大腦筆記庫 (如 Obsidian)」。只有寫下來的，才是唯一真實的知識資產。

AI 閱讀三大心法（預告）

🏹 狩獵法

帶問題讀資料，不求讀完，只求破局

📝 費曼法

讓 AI 反問你，逼你用自己的話解釋

⚔️ 辯論法

讓不同觀點辯論，找出最佳方案

💡 我們在第六堂課會深入學習這三種心法。今天先體驗「費曼法」！

第一步：設備準備

  • 電腦（Mac / Windows / Linux 皆可）
  • 穩定的網路連線
  • 終端機（Mac：終端機 / Windows：PowerShell）
  • 瀏覽器（Chrome / Safari / Edge）
  • 手機（用於測試 Telegram / Discord）

終端機在哪裡？

Mac

應用程式 → 工具程式 → 終端機

或按 ⌘ + 空白鍵，搜尋「終端機」

Windows

按 Windows 鍵，搜尋 PowerShell

或按 Win + R，輸入 powershell

第二步：帳號申請

  • Telegram — 下載 App，註冊帳號
  • OpenRouter — 前往 openrouter.ai 註冊
  • Zeabur — 前往 zeabur.com 用 Google 登入
  • GitHub（可選）— 前往 github.com 註冊

申請 OpenRouter API Key

  1. 前往 openrouter.ai
  2. 點擊「Sign Up」註冊帳號
  3. 登入後，點擊左側「Keys」
  4. 點擊「Create Key」建立新金鑰
  5. 複製金鑰（格式：sk-or-v1-...）
  6. 妥善保存！不要分享給別人
💡 OpenRouter 提供 $1 免費額度，夠你用好幾天！

🛡️ AI 特助的安全防線

AI 具有執行權限，安全自律是開發的第一守則

1. 權限邊界：半自動 vs 全自動

新手務必使用「半自動確認機制」。當 AI 執行命令（如讀寫、終端指令）時，仔細審查後才點擊核准。嚴禁隨意 Bypass，防止 AI 幻覺誤刪系統檔案。

2. 金鑰防禦：絕對不開金鑰後門

不要在任何對話中直接傳送 API Key 或帳號密碼給 AI，更不要將金鑰 commit 到公開 GitHub。金鑰必須存放在本地 .env 環境變數中！

動手做：防禦性專案初始化

在開始與 AI 協作前，為你的目錄裝上安全護欄

📂 1. 建立專案目錄與 Git 初始化 — 打開終端機輸入：
mkdir my-ai-project && cd my-ai-project
git init
🛡️ 2. 配置防禦性 .gitignore — 建立並編輯 .gitignore，防止敏感檔案外流：
# 寫入以下過濾規則
.env
*key*
*.json
.DS_Store
💡 這樣即使 Agent 自動執行 Git 提交，也絕對不會把您的 OpenRouter Key 或是敏感憑證上傳到 GitHub！

動手做：費曼法體驗

體驗完整 AI 協作閉環（開工 → 費曼法對話 → 收工）

🌅 1. 開工對齊 — 對 Bot 發送：
開工！今天我要用費曼法練習解釋『AI 助理』。請扮演蘇格拉底式的思維教練。用最簡單、連六歲小孩都能聽懂的邏輯向我追問，先讓我用一句話解釋『什麼是 AI 助理』，並找出我理解中的漏洞，挑戰我的結論。
🧠 2. 費曼法對決 — 跟著 Bot 的追問，用你自己的白話回答它，直到能完美解釋。
🌌 3. 收工存檔 (SSOT) — 對話結束時發送：
收工！請幫我整理今天理解的重點與盲點到我的 SSOT 小卡中，並清理暫存。 *別忘了將產出複製到你的實體卡片或筆記中！

為什麼費曼法有效？

一般問法

問 AI 答案 → 看完就忘 → 下次還是不會

費曼法

AI 反問你 → 你被迫思考 → 真正搞懂

💡 如果你習慣用 AI 來替你思考，你會變得越來越不會思考
如果你用 AI 來當思維教練，你會變得比之前聰明十倍
