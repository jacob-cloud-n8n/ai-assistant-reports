# AI 助理實戰課程 — 教材目錄

本頁僅公開課程教材目錄與內容概覽。完整教材、部署 SOP、提示詞模板及工作規則檔，於簽約後依課程進度交付，不在公開網站散布。

| 模組 | 主題 | 主要成果 |
|---|---|---|
| M0 | 課程導覽與環境準備 | 確認設備、帳號與安全原則 |
| M1 | AI 指令素養 | 建立有效溝通與問題拆解能力 |
| M2 | AI 失憶與第二大腦 | 建立可持續維護的知識架構 |
| M3 | Agent 選擇決策 | 依任務挑選合適的 AI Agent |
| M4 | 第一個行動助理 | 完成 Telegram／Discord 基礎部署 |
| M5 | 隨身書僮實戰 | 建立持久化與備份機制 |
| M6 | 五人 Agent 團隊 | 定義分工、交接與審核流程 |
| M7 | Cron 排程與管理 | 建立定時任務與狀態管理 |
| M8 | Agent 團隊規劃 | 畫出權責與資訊流藍圖 |
| M9 | AI 閱讀三大心法 | 實作狩獵法、費曼法與辯論法 |
| M10 | 踩坑與排錯 | 建立可重用的故障排除紀錄 |
| M11 | 成本試算與優化 | 管理模型用量與運作成本 |
| M12 | 安全實踐 | 建立金鑰、權限與資料隔離檢查 |

方案 B 共 13 堂核心建置課，每堂 2 小時，總計 26 小時。
