為什麼要學 AI 素養？

多數人只會問「幫我寫...」，學完差別巨大

之前之後

提示詞公式

角色 + 任務 + 格式 + 限制

💡 範例：
「你是資深行銷專家（角色），幫我寫一封活動邀請 email（任務），用 150 字以內、分三段（格式），語氣要專業但親切（限制）」

AI 閱讀三大心法（預告）

🏹 狩獵法

帶問題讀資料，不求讀完，只求破局

📝 費曼法

讓 AI 反問你，逼你用自己的話解釋

⚔️ 辯論法

讓不同觀點辯論，找出最佳方案

💡 我們在第六堂課會深入學習這三種心法。今天先體驗「費曼法」！

第一步：設備準備

  • 電腦（Mac / Windows / Linux 皆可）
  • 穩定的網路連線
  • 終端機（Mac：終端機 / Windows：PowerShell）
  • 瀏覽器（Chrome / Safari / Edge）
  • 手機（用於測試 Telegram / Discord）

終端機在哪裡？

Mac

應用程式 → 工具程式 → 終端機

或按 ⌘ + 空白鍵，搜尋「終端機」

Windows

按 Windows 鍵，搜尋 PowerShell

或按 Win + R，輸入 powershell

第二步：帳號申請

  • Telegram — 下載 App，註冊帳號
  • OpenRouter — 前往 openrouter.ai 註冊
  • Zeabur — 前往 zeabur.com 用 Google 登入
  • GitHub（可選）— 前往 github.com 註冊

申請 OpenRouter API Key

  1. 前往 openrouter.ai
  2. 點擊「Sign Up」註冊帳號
  3. 登入後，點擊左側「Keys」
  4. 點擊「Create Key」建立新金鑰
  5. 複製金鑰（格式：sk-or-v1-...）
  6. 妥善保存！不要分享給別人
💡 OpenRouter 提供 $1 免費額度，夠你用好幾天！

動手做：費曼法體驗

讓 AI 反問你，逼你用自己的話解釋

📝 現在打開 Telegram，對你的 Bot 說：

「請扮演蘇格拉底式的思維教練。用最簡單、連六歲小孩都能聽懂的邏輯不斷向我追問：
1. 先讓我用一句話解釋「什麼是 AI 助理」
2. 找出我理解中的漏洞，挑戰我的結論
3. 直到我能用最白話的方式完美解釋為止」

為什麼費曼法有效？

一般問法

問 AI 答案 → 看完就忘 → 下次還是不會

費曼法

AI 反問你 → 你被迫思考 → 真正搞懂

💡 如果你習慣用 AI 來替你思考，你會變得越來越不會思考
如果你用 AI 來當思維教練，你會變得比之前聰明十倍
