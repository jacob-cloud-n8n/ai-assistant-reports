## 🗺️ 您將完整解鎖的「13 大教學建置模組」

我們將在 **13 堂課中（每堂課 2 小時，總計 26 小時。可視當天實作進度進行彈性延長或調整成為課後 QA 時間，不另行加收費用或退費）**，帶您循序漸進解鎖並親手建置以下 13 個核心教學模組，徹底理清您的 AI 工作流：

* **模組 1：AI 指令素養與環境準備** ➔ 建立正確的 Prompt 互動心法，完成本機 Terminal、軟體工具與 AI 帳號註冊。
* **模組 2：AI 的失憶症與機制剖析** ➔ 深入剖析 AI 脈絡限制，理解為什麼每次都要重新對話，以及如何解決此痛點。
* **模組 3：建立個人數位第二大腦** ➔ 規劃「輸入 ➔ 消化 ➔ 輸出」三層結構，建置結構化 Markdown 本地知識庫。
* **模組 4：Agent 團隊全覽與選型決策** ➔ 釐清各個 AI Agent 的特質、極限，學會如何挑選最適合任務的專屬助理。
* **模組 5：動手做：安裝設定您的第一個 Agent** ➔ 實地安裝配置行動助理 Hermes，完成首次連線與基本對話調校。
* **模組 6：實戰：手機隨身書僮部署與 Volume 設定** ➔ 部署 Discord/Telegram 雙平台助理，建置儲存磁碟完成防失憶持久設定。
* **模組 7：實戰：五人 Agent 團隊協作** ➔ 實作多個獨立 Agent 在同一個專案中分工協同開發，體驗 AI 團隊真實工作流。
* **模組 8：從使用者升級為 AI 管理者** ➔ 實作 Cron 定時自動化排程，設定助理的短期與長期記憶檔案。
* **模組 9：Agent 團隊架構藍圖規劃** ➔ 繪製專屬 AI 團隊組織架構藍圖，定義各 Agent 的權責、溝通鏈與輸入輸出規範。
* **模組 10：AI 智慧閱讀三大心法實裝** ➔ 實裝並熟練「狩獵法、費曼法、雙腦辯論法」，用 AI 進行超高效知識吸收。
* **模組 11：技術踩坑紀錄與排錯 SOP** ➔ 整理開發與維護過程中常見的錯誤代碼與狀態，建立本機排錯自檢流程。
* **模組 12：模型成本試算與用量優化術** ➔ 進行 API Token 的用量精算，制定最佳的調度策略，省下高達 80% 的運行成本。
* **模組 13：安全實踐與資安防禦屏障** ➔ 設定敏感資訊與 API Key 的自動化過濾機制，確保機密資料絕不上傳雲端。
