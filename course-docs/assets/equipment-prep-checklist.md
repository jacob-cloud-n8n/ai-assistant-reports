## 📋 一、 帳號準備區

- [ ] **OpenRouter 帳號註冊**
  - **說明**：OpenRouter 是一個強大的 LLM API 聚合平台，能讓您以極低的成本使用各種頂尖模型（如 Claude 3.5 Sonnet、Gemini 1.5 Pro 等）。
  - **路徑**：前往 [OpenRouter 官網](https://openrouter.ai/) 註冊。
  - **免費額度**：註冊後可直接使用其提供的免費模型（例如帶有 `free` 標記的模型），或儲值低至 $5 美元即可開始使用付費 API。

- [ ] **Telegram 帳號與應用程式**
  - **說明**：用於建立與測試您的個人 Telegram 隨身特助。
  - **步驟**：在手機及電腦上安裝 Telegram，並確認能正常登入使用。

- [ ] **Zeabur 雲端部署帳號**
  - **說明**：Zeabur 是一個一鍵部署平台，能幫我們把寫好的 Bot 部署到雲端，24 小時不中斷運行。
  - **路徑**：前往 [Zeabur 官網](https://zeabur.com/)。
  - **登入方式**：建議直接使用 **Google 帳號** 或 **GitHub 帳號** 登入。

## 🐚 三、 命令列基本指令速查 (Command Line Cheat Sheet)

在後續的實作中，我們偶爾會需要打開「終端機」(Terminal) 輸入指令。請不用害怕，您只需要理解以下幾個最常用的指令即可：

| 指令 | 英文全名 | 白話意思 | 實際範例與用法 |
| :--- | :--- | :--- | :--- |
| `pwd` | **P**rint **W**orking **D**irectory | 「我現在在哪裡？」顯示目前目錄的絕對路徑。 | 輸入 `pwd` 回車，會看到類似 `/Users/yourname/Projects` 的路徑。 |
| `ls` | **L**i**s**t | 「這裡有什麼？」列出當前目錄下的所有檔案與資料夾。 | 輸入 `ls` 可以看見當前資料夾內的檔案清單。 |
| `cd` | **C**hange **D**irectory | 「去某個地方。」切換到指定的資料夾。 | `cd Desktop` 代表進入桌面；`cd ..` 代表回到上一層資料夾。 |
| `mkdir` | **M**ake **D**irectory | 「生一個新資料夾。」建立一個新的目錄。 | `mkdir my-bot` 會在當前路徑下建立一個名為 `my-bot` 的資料夾。 |
| `git init` | **Git** **Init**ialize | 「讓 Git 開始幫我記錄。」在當前資料夾初始化一個 Git 版本控制庫。 | 在您的專案資料夾內輸入 `git init`，為後續的備份與備份做準備。 |
