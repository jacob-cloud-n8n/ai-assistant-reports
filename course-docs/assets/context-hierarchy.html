Custom Instructions / 全域 System Prompt

L3 · 當前對話臨時 Prompt (最高優先)

您在聊天視窗直接發送的指令。具備最高覆蓋權限，用來應付「這一次」的特殊需求（例：「這封信請用英文寫，不用中文」）。

L2 · 專案說明書 CLAUDE.md (中等優先)

儲存於特定專案資料夾下的規範。當進入該目錄時自動載入，會覆蓋 L1 全域設定。適合規範特定專案的寫作風格或代碼規範。

L1 · 系統說明書 (基礎底層)

AI 的全域基礎個性與偏好（如 Custom Instructions）。這是在所有對話中預設生效的底層指令，任何特殊規則都建立在此基礎之上。
