## 第 4 堂：M3 Agent 全覽 — 選對的工具

### 頁 1：封面
**標題**：Agent 全覽 — 選對的工具
**要點**：
- 不是聊天機器人，是有記憶、會學習、能執行的助理
- 市面 Agent 百百種，選錯工具 = 浪費時間

### 頁 2：Agent 是什麼
**標題**：Agent ≠ 聊天機器人
**要點**：
- ChatGPT 只能對話，每次開新對話從零開始
- Agent 有記憶、能執行指令、能連接外部工具
- 想像：聊天機器人是「問答機」，Agent 是「隨身助理」

### 頁 3：市面常見 Agent 比較
**標題**：四大 Agent 一次看
**要點**：
- Hermes：隨身助理，記憶＋排程，適合個人日常
- Codex：工作流助理，執行＋自動化，適合開發者
- OpenCode：遠端執行，適合需要伺服器的場景
- n8n：自動化中樞，串接多個服務，適合進階用戶

### 頁 4：如何選擇
**標題**：依需求選 Agent
**要點**：
- 個人日常 → Hermes（Telegram/Discord）
- 開發自動化 → Codex + OpenCode
- 多服務串接 → n8n
- 預算有限 → 先從免費方案開始

### 頁 5：實做 — 決策表
**標題**：找出最適合你的 Agent 組合
**要點**：
- 填寫你的需求（日常/開發/自動化）
- 依裝置選擇（手機/電腦/伺服器）
- 依預算選擇（免費/付費）
- 學員帶走：`Agent選擇決策表.md`

## 第 5 堂：M4 動手做 — 你的第一個 Agent

### 頁 1：封面
**標題**：動手做 — 你的第一個 Agent
**要點**：
- 今天你會完成 Hermes 安裝
- 跟你的 AI 助理說第一句話

### 頁 2：部署選擇
**標題**：Telegram vs Discord
**要點**：
- Telegram：推薦一般用戶，簡單快速
- Discord：推薦家庭教育，可建私人伺服器
- 兩者都支援語音、圖片、檔案傳輸

### 頁 3：Hermes 安裝
**標題**：一行指令安裝
**要點**：
- 安裝 Hermes（一行指令）
- 設定環境變數（API Key）
- 首次啟動（`/start`）

### 頁 4：首次對話
**標題**：跟你的 AI 說第一句話
**要點**：
- 發送 `/start` 啟動
- 測試基本功能（問問題、傳圖片）
- 確認記憶正常（問它「你記得我嗎？」）

### 頁 5：實做 — 完成安裝
**標題**：動手做時間
**要點**：
- 依選擇的介面完成部署
- 與你的 AI 助理說第一句話
- 學員帶走：`Hermes開工.md`、`Hermes收工.md`
