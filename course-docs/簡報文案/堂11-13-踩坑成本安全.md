## 第 11 堂：M10 踩坑紀錄與防呆

### 頁 1：封面
**標題**：踩坑紀錄與防呆
**要點**：
- 這些是真實發生過的事故
- 讀完你就知道「哪些事不能做」

### 頁 2：常見踩坑
**標題**：六大踩坑
**要點**：
- Token 外洩（API Key 被偷）
- 長對話變笨（上下文塌陷）
- 自動化誤觸發（測試搞到 production）
- 同 Bug 三次喊停（燒額度）
- 部署到錯誤專案（搞混環境）
- 制度自己違反自己（自律失效）

### 頁 3：解法速查
**標題**：每個問題的快速解法
**要點**：
- Token 外洩 → 廢棄舊 Key、產生新 Key、檢查盜用
- 長對話變笨 → 壓縮寫回、開新對話
- 自動化誤觸發 → 測試用獨立環境、Jacob 拍板
- 同 Bug 三次 → 凍結、等 Jacob 決策
- 部署錯誤 → 確認目錄、用腳本鎖定
- 制度違反 → 寫成腳本自動檢查

### 頁 4：實做 — 走讀清單
**標題**：動手做時間
**要點**：
- 走讀踩坑清單
- 標記你遇過的
- 記錄你的解法
- 學員帶走：`事故案例集v1.md`、`止損三次卡.md`

## 第 12 堂：M11 成本優化術

### 頁 1：封面
**標題**：成本優化術
**要點**：
- 今天你會學省錢策略
- 試算你的 AI 月費

### 頁 2：成本結構
**標題**：AI 花多少錢？
**要點**：
- 工具費用（免費/付費）
- API 費用（按 Token 計費）
- 部署費用（Zeabur/自架）

### 頁 3：省錢策略
**標題**：三個省錢大法
**要點**：
- 機械檢查給腳本（零 Token）
- LLM 只做判斷（省 90%）
- 用便宜模型做粗工（B 級），聰明模型做判斷（S 級）

### 頁 4：實做 — 試算
**標題**：動手做時間
**要點**：
- 計算你的預期用量
- 找出最省錢的方案
- 學員帶走：`省token分工卡.md`

## 第 13 堂：M12 安全實踐

### 頁 1：封面
**標題**：安全實踐
**要點**：
- 今天你會完成安全檢查清單
- 保護你的 AI 助理和資料

### 頁 2：安全威脅
**標題**：三大威脅
**要點**：
- API Key 外洩（被盜用）
- 未授權存取（被別人用）
- 資料洩漏（被別人看）

### 頁 3：防護措施
**標題**：三道防線
**要點**：
- 白名單機制（只有授權的人能用）
- 金鑰過濾（`.gitignore` + 敏感掃描）
- 資料隔離（`.env` 不進 Git）

### 頁 4：實做 — 安全檢查
**標題**：動手做時間
**要點**：
- 走讀安全清單
- 確認你的設定
- 學員帶走：`安全檢查清單.md`
