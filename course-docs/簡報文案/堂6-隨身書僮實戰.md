## 第 6 堂：M5 實戰案例 — 隨身書僮

### 頁 1：封面
**標題**：實戰案例 — 隨身書僮
**要點**：
- 今天你會建立自己的 Discord Bot
- 完成 Zeabur 部署
- 學會防失憶 Volume 設定

### 頁 2：為什麼選 Discord
**標題**：Discord 的優勢
**要點**：
- 免手機門號（Telegram 需要）
- 可建私人伺服器（團隊协作）
- 適合多人使用（家庭教育）
- 豐富的權限管理（角色、頻道、存取控制）

### 頁 3：隨身書僮是什麼
**標題**：不只是聊天機器人
**要點**：
- 隨身書僮 = 有記憶的 AI 助理，24hr 在線
- 它會記得你說過的話、你的偏好、你的專案進度
- 你可以跟它對話、傳檔案、請它幫忙做事
- 真實案例：一位創業者用隨身書僮管理團隊、記帳、排程

### 頁 4：今天的流程
**標題**：四步完成
**要點**：
1. 建立 Discord Bot（Developer Portal）
2. 設定 Zeabur 部署
3. 環境變數與 Volume 設定
4. 測試對話與記憶功能

### 頁 5：Step 1 建立 Discord Bot
**標題**：Developer Portal 操作
**要點**：
- 進 Discord Developer Portal（`discord.com/developers`）
- New Application → 取名（例如「我的書僮」）
- 左側選 Bot → Reset Token → 複製 Token
- 開啟 MESSAGE CONTENT INTENT（重要！不開會讀不到訊息）

### 頁 6：Step 2 邀請 Bot 到伺服器
**標題**：OAuth2 邀請連結
**要點**：
- 左側選 OAuth2 → URL Generator
- Scopes 勾選 `bot`
- Bot Permissions 勾選：Send Messages、Read Messages、Attach Files
- 複製連結 → 貼到瀏覽器 → 選擇你的伺服器 → 授權
- 驗證：伺服器出現你的 Bot = 成功

### 頁 7：Step 3 Zeabur 部署
**標題**：一鍵部署
**要點**：
- 進 Zeabur → 匯入 Hermes 模板
- 填入 Discord Token（環境變數 `DISCORD_TOKEN`）
- 填入白名單（`DISCORD_ALLOWED_USERS`，你的 Discord User ID）
- 部署 → 等 30 秒 → 看到 "Bot is running"

### 頁 8：Step 4 找到你的 User ID
**標題**：怎麼拿到白名單 ID
**要點**：
- 開啟 Discord 設定 → 進階 → 開啟「開發者模式」
- 回到伺服器 → 右鍵你的頭像 → 「複製使用者 ID」
- 貼到 `DISCORD_ALLOWED_USERS`
- 沒有這個 ID = Bot 不理你（安全機制）

### 頁 9：Volume 設定
**標題**：防失憶
**要點**：
- 問題：Zeabur 容器重啟會丟資料（記憶歸零）
- 解法：掛載 Volume 到 `/root/.hermes`
- 操作：Zeabur → 你的服務 → Volumes → 新增 → 路徑填 `/root/.hermes`
- 驗證：重啟服務 → 問 Bot「你記得我嗎？」→ 如果記得 = 成功

### 頁 10：首次對話
**標題**：跟你的書僮說第一句話
**要點**：
- 在 Discord 頻道 @你的 Bot 或私訊它
- 發送：「你好，我是你的主人」
- 它會回你問候
- 記憶測試：先說「我喜歡吃拉麵」→ 過一陣子問「我喜歡吃什麼？」

### 頁 11：基本功能測試
**標題**：玩玩看
**要點**：
- 問問題：「幫我整理今天的待辦事項」
- 傳圖片：上傳一張照片請它描述
- 傳檔案：上傳一份 PDF 請它摘要
- 記憶：告訴它「我的專案叫鮮果樂園」→ 過一陣子問「我的專案叫什麼？」

### 頁 12：安全機制
**標題**：白名單 + 權限管理
**要點**：
- **白名單**：只有 `ALLOWED_USERS` 裡的人能跟 Bot 對話
- **權限管理**：限制 Bot 能做什麼（例如不能刪除檔案）
- **Token 安全**：絕不把 Token 貼到公開地方
- **定期更換**：建議每 3 個月換一次 Token

### 頁 13：常見問題
**標題**：卡住了怎麼辦
**要點**：
- Bot 不回話 → 檢查 Token、ALLOWED_USERS、MESSAGE CONTENT INTENT
- 記憶消失 → 檢查 Volume 有沒有掛載
- 回覆很慢 → 免費方案正常，升級可加速
- 錯誤訊息 → 貼到 Google 搜尋

### 頁 14：進階：自訂人格
**標題**：讓書僮更有個性
**要點**：
- 你可以告訴它「請用溫柔的語氣跟我說話」
- 設定它的專業領域：「你是我的財務助理，專門記帳」
- 建立工作流：「每天早上 9 點提醒我今天的行程」
- 進階玩法：串接 Notion、Google Calendar、n8n

### 頁 15：實做 — 完成部署
**標題**：動手做時間
**要點**：
- 建立 Discord Bot
- 完成 Zeabur 部署
- 設定 Volume 防失憶
- 測試對話與記憶功能
- 學員帶走：`Discord部署SOP.md`、`Telegram部署SOP.md`

### 頁 16：小結
**標題**：M5 小結——你的書僮上線了
**要點**：
- **四步部署**：建 Bot → 邀請 → Zeabur → 測試
- **防失憶**：Volume 掛載到 `/root/.hermes`
- **安全**：白名單 + Token 保護
