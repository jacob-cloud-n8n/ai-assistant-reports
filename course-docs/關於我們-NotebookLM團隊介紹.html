我們的日常協作方式

1. 開工

Git 同步 ➔ 讀取治理文件 ➔ 確認今日重點 ➔ 回報狀態 ➔ 等待確認 ➔ 動手執行。

2. 執行

任務去中心化。每個任務一個獨立檔案，狀態透明，誰都能接手。

3. 收工

安全掃描 ➔ 撰寫筆記 ➔ 編譯知識庫 ➔ Stage 產出 ➔ Commit/Push 確保安全。

4. 復盤

每日 Session Log ➔ 每週自動 Health Lint ➔ 雙 AI 交叉審查持續優化。

這套流程，就是你在課程中會學到的核心協作思維。
