## 故事線 v0

### 序幕（一個選擇）✅ Jacob 2026-07-09 口述
原本在學校工作。老爸年紀大了，問我有沒有意願回來試試銷售羊奶。看著學校裡升遷無望的路，想著至少時間彈性、還能幫家裡，我選擇回來。

### 目標（曾經的渴望）✅ Jacob 2026-07-09 口述
當初接下羊奶的工作，想的是：時間可以自己安排、多點時間陪孩子長大、放假帶家人出去走走。裸辭之後，帶著老婆和孩子去南部玩了三天——那時候沒有收入，但我心裡很清楚：**只要肯努力，這樣的日子不會是最後一次。** 我要的不是變有錢，是一家人想出門就能出門。

> 簡報頁金句候選：「沒收入的那三天，是我最想留住的生活。」

### 阻礙（最痛的時刻）✅ Jacob 2026-07-09 口述
創業教會我的第一件事是：**老闆比員工更沒有自己的時間。** 想靠假日陪家人？假日正是服務業最不能錯過的營業日。當初為了時間自由離開學校，結果連「放假」兩個字都從字典裡消失了。

回來的第一年很痛苦。靠失業補助過生活；補助結束那天，生意雖然有點起色，但老婆全職在家帶兩個孩子，完全入不敷出。羊奶這行是**有做就有收入，沒做就沒有**——身體就是產能，停一天就少一天的錢。

> 簡報頁金句候選：「為了時間自由創業，結果假日成了最忙的營業日。」

最痛的一次：搬羊奶桶（約 50 公斤）閃到腰，痛到沒辦法工作，還是撐著出門送完。回到家原本想躺一下，一抬頭看到桌上還沒繳的帳單——又硬著頭皮拿起電話，一個一個問客戶：「還有沒有要送羊奶的？」希望能再多掙一點。

> 簡報頁金句候選：「腰可以閃到，帳單不能遲到。」／「那時候我的『事業』，就是我的腰。」

### 努力（三波嘗試，前兩波都跌倒）✅ Jacob 2026-07-09 口述

**第一波：花錢買教訓。** 為了省錢買二手電腦，結果跑不動 AI 工具，只好再買一台新的；又在鏡頭、麥克風等直播設備上花了不少冤枉錢——後來才發現，手機拍攝＋鏡頭前的朗讀練習就很夠用了。

**第二波：代理夢碎。** 為了幫店裡生意，代理了美國的網站系統——想著線上銷售能幫事業成長，賣網站還能多一份斜槓收入。結果每個月都有不小的資金支出，跟美國端的聯繫又不順暢，出了問題我被夾在公司跟客戶中間，**兩邊不是人**。

**第三波：自己來，撞牆撞到無力。** 發現 AI agent 或許能幫我架網站，就結束代理權、開始用 agent 做網站——第一個客戶就是自己，幫小牧人建站。因為不熟悉，初版網站只有表面：想改局部，整個頁面被改掉；想加個新配色，出來變成另一個網站；更別說怎麼串後端、之後怎麼更新。看了很多教學影片，全是零散的知識片段，勉強組裝起來，一實做才發現**我的操作畫面跟教學的根本不同**——無力感真的很重。

**轉折：不學寫程式，改學「怎麼管 AI」。** 從一個 AI 助理開始，踩坑、留紀錄、立規矩——每次失敗都變成一條制度，慢慢長成一支 6 個 AI agent 的團隊：有人寫程式、有人做設計、有人管自動化、有人隨身待命。

> 簡報頁金句候選：「改一個角落，毀一整頁。」／「教學影片看了幾十支，畫面跟我的就是不一樣。」／「我沒有學會寫程式，我學會了當 AI 的老闆。」

### 結果（那個早上，訂單自己來了）✅ Jacob 2026-07-09 口述
轉機出現在一次聚餐：朋友推薦了 n8n，分享了高健龍的教學影片。學的過程還是會遇到畫面跟教學不一樣的地方，但這次我有了新方法——**截圖問 Gemini**，來來回回地磨，總算串起小牧人的 LINE 官方帳號。這是我第一次把工作流跟 agent 接起來，讓它幫我回覆客戶的問題。

然後是那個早上。我照常出門送羊奶，手機傳來通知：**有一筆新訂單。** 後來一看才知道——客戶半路傳訊息來問，自動化的小牧人及時回覆了，客戶當場下單訂了羊奶。**我人在路上送奶，我的「員工」在店裡接單。**

從那一刻起，我知道 agent 真的能用，開始把它導入所有工作。今天，我的角色只剩「拍板」：6 個 AI agent 自己開工、自己交接、自己驗收——官網更新、課程營運、案例上架，一條一條變成自動產線。原本一個人做不完的事，變成一句話就派出去。

而這整套系統本身，就是我們教的東西——我們不是教理論，我們每天靠它過日子。當年南部的那三天，現在隨時可以再來。

> 簡報頁金句候選：「我在送羊奶，我的員工在接單。」／「那個早上，訂單自己來了。」

### 使命（從做給你，到教會你）✅ Jacob 2026-07-09 口述
接下來，我從身邊的人開始測試 agent 到底能做到什麼程度：

1. **自己**：把那個半殘的網站救回來。
2. **收納師（太太）**：建新網站＋LINE 官方助理，她在案場忙碌時，有個小助理幫忙回 QA、收訂單。
3. **咖啡店**：設定進銷存系統，老闆不用再在忙碌中處理一堆報表，可以專心烘豆。

一個個任務完成之後，我反而想通了一件事：**我能做的實在太有限。** 客戶把問題轉述給我、我來設計解決方案——這中間隔了一層；還不如他直接跟 agent 討論，我只要教會他使用 agent 就行。

因為這個想法，我開始規劃家教班課程。第一個目標很具體：**協助 10 位客戶學會 agent，用它解決自己工作上的痛點。** 後續的線上課程，就是把這件事放大。

> 簡報頁金句候選：「與其替你解決，不如教你指揮。」／「我能做的有限，能教的沒有上限。」
