Agent ≠ 聊天機器人

對話只是手段，能調用工具並主動達成目標才是真正的 Agent

· ChatGPT 只能對話，每次開新對話從零開始

· Agent 有記憶（跨對話記得你）、能執行指令（不只說還會做）、能連接外部工具（串 API、讀檔案、跑腳本）

· 想像：聊天機器人是「問答機」，Agent 是「隨身助理」

運作模式對比
聊天機器人 (Chatbot)被動回答、無外部工具、開新對話即失憶
AI 特助 (Agent)主動執行、連接硬碟與 API、擁有長效記憶
