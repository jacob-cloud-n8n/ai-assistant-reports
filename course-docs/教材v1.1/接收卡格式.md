# 📥 接收卡格式 — 一人版小卡

> 派工給 AI 時，用這張卡讓它「一眼看懂要做什麼」。

---

## 接收卡模板

```
📥 **接收卡：{任務名}**

{一句話任務描述}

第一優先：{具體行動}
第二優先：{如有}

任務檔：`.tasks/{filename}.md`
```

---

## 好的接收卡 vs 壞的接收卡

| ✅ 好的 | ❌ 壞的 |
|---------|---------|
| 明確收件者（只有一個 agent） | 一張卡發給多個 agent |
| 一句話任務描述 | 只寫「請看任務檔」 |
| 排優先的行動項目 | 混在長篇通知裡 |
| 附任務檔路徑 | 沒有具體行動 |

---

## 範例

```
📥 **接收卡：官網 FAQ 頁面製作**

幫官網新增「常見問題」頁面，收錄 10 個最高頻提問。

第一優先：訪談客服團隊，整理 top 10 問題與標準回答
第二優先：設計頁面版型，確保手機端可讀

任務檔：`.tasks/faq-page-design-2026-07-15.md`
```
