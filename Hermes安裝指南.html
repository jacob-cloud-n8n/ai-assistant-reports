開始使用

步驟一：建立 Telegram Bot

1. 開啟 Telegram，搜尋 @BotFather
2. 發送指令：
/newbot

3. 輸入機器人名稱（如：我的AI助理）
4. 輸入機器人帳號（如：my_ai_assistant_bot）

5. 記錄獲得的 Bot Token
⚠️ 請妥善保管，不要分享給他人

步驟二：Zeabur 一鍵部署

1
登入 Zeabur
前往 zeabur.com，使用 Google 或 GitHub 帳號登入
2
新增服務
點選「新增服務」→「模板」
3
搜尋 Hermes Agent
選擇第一個模板（約 1300+ 次部署）
4
確認部署
點選「確認」，等待 1-2 分鐘

步驟三：設定環境變數

在 Zeabur 服務 → 環境變數頁面新增

變數名稱 值 說明
TELEGRAM_BOT_TOKEN 你的 Bot Token 從 BotFather 取得
HERMES_INFERENCE_PROVIDER deepseek LLM 供應商
HERMES_INFERENCE_MODEL deepseek-v4-flash 模型名稱
HERMES_INFERENCE_API_KEY 你的 API Key 從 DeepSeek 平台取得
💡 可選設定：ALLOWED_TELEGRAM_USER_IDS 限制只有特定用戶可以使用

步驟四：驗證部署

1. 檢查服務狀態
在 Zeabur 後台確認服務狀態為「運作中」
2. 測試 Telegram
在 Telegram 搜尋你的機器人，發送訊息測試
3. 檢查日誌
在 Zeabur「日誌」頁面確認無錯誤訊息
4. 健康檢查
瀏覽器訪問 https://你的網址/health

自架部署方式

一鍵安裝

適用 Linux / macOS / WSL2
Docker Compose

需要 Docker 環境
