為什麼要學 AI 素養？

大多數人只把 AI 當玩具隨意問，改用語境引導能獲得數倍的回答品質

情境 一般人問法（得過且過） 高手自學問法（精準破局）
寫 Email 「幫我寫一封回覆詢價的信。」 「你是資深行銷專家，請幫我寫一封回覆詢價的 Email，字數限制 150字內、分三段，語氣要親切專業...」
尋求解答 「這個怎麼做？」 「我目前在 [情境]，遇到 [具體阻塞點]，你覺得最符合效率的解決步驟是什麼？」
概念學習 「什麼是 API？」 「請使用費曼法，用連六歲小孩都聽得懂的比喻向我解釋 API，並在結尾提出問題確認我懂了。」

提示詞黃金公式

角色 (Role) + 任務 (Task) + 格式 (Format) + 限制 (Constraint)

萬用公式示範
「你是資深時間管理教練 (角色)，幫我設計一份遠端工作者的每日作息表 (任務)。請使用Markdown 表格輸出 (格式)，限制排除晚餐後的任何工作安排，並加入番茄鐘休息區間 (限制)。」
💡 自學提示：每次與 AI 互動前，花 5 秒對齊這四個要素，就能免去 80% 的來回校正時間。

第一步：硬體與終端機準備

  • 準備電腦（Mac / Windows / Linux 均可，建議安裝 Chrome 瀏覽器）
  • 學會開啟終端機（Terminal）作為控制台

🍏 macOS 終端機

點選「應用程式」→「工具程式」→「終端機」；或按下 Command + 空白鍵 搜尋「終端機」。

🪟 Windows PowerShell

按下鍵盤 Win 鍵，輸入「PowerShell」開啟；或按下 Win + R 輸入 powershell。

第二步：雲端服務帳號申請

請跟著課程步驟，完成以下服務的註冊：

  • Telegram：下載 App 並註冊，這將是您的 AI 特助互動通道。
  • OpenRouter：前往 openrouter.ai，使用 Google 帳號一鍵註冊取得 API Key。
  • Zeabur：前往 zeabur.com，提供雲端一鍵部署服務的平台。
  • GitHub：前往 github.com 註冊，用於同步您的第二大腦與程式碼。

OpenRouter 金鑰申請步驟

  1. 打開瀏覽器，導航至 openrouter.ai。
  2. 點擊右上角的 Sign In / Sign Up，選擇 Google 登入。
  3. 登入後，點擊左側選單的 Keys，再點擊右上角 Create Key。
  4. 為金鑰命名（例如：course-key），並點擊確認。
  5. 重要！ 複製以 sk-or-v1- 開頭的完整金鑰，並暫存在本地安全的文字檔。
⚠️ 安全提示：OpenRouter 金鑰生成後只會顯示一次。千萬不要將它截圖、發送到公開聊天室，或直接上傳到 GitHub！

🛡️ AI 特助的安全防禦

AI 擁有讀寫本機與雲端的權限，安全自律是開發協作的核心

1. 半自動確認機制

當 AI 助理欲執行寫入或終端機指令時，務必人工審查後才放行。拒絕一味點選「Bypass」，避免幻覺指令導致的檔案刪除風險。

2. 金鑰環境變數隔離

絕不將明文 API Key 寫死在代碼中，應統一存放在 .env 檔案。並確保 .gitignore 已排除該檔案，防止金鑰推送至公開倉庫。

動手做：防禦性專案初始化

為你的第一個 AI 專案建立安全圍欄，避免上傳金鑰：

📂 1. 建立目錄並初始化 Git (終端機操作)：
mkdir my-ai-project && cd my-ai-project
git init
🛡️ 2. 建立 .gitignore 檔案 (寫入以下內容並存檔)：
.env
*key*
*.json
.DS_Store

AI 協作自律禮儀：開工與收工

AI 助理不是一問一答的搜尋框，而是你的工作夥伴

🌅 1. 開工對齊 (Start Work)

每次開啟新對話時，輸入「開工」Prompt，讓 AI 自動載入任務設定、專案目標與目前進度，防範上下文遺忘，免去反覆背景解釋。

🌃 2. 收工整理 (Finish Work)

對話結束前輸入「收工」，讓 AI 總結今日產出、標明剩餘卡點、排除任何隱私或金鑰資料，確保下次開工能無縫接軌。

單一真相來源 (SSOT)

對話視窗是沙灘上的腳印，隨時會被潮水沖走

Single Source of Truth (SSOT) 原則：
由於 AI 的對話記錄與記憶隨時可能隨着對話清理或額度耗盡而蒸發，重要的程式碼片段、決策、或是你的學習重點，一律禁止只存留在對話記錄中！必須在收工時，將其同步整理至您的「第二大腦 (Obsidian 筆記庫)」或實體的「黃金收工卡」中。

自學高效心法：費曼學習法

如果習慣只向 AI 要答案，大腦會逐漸失去思考與解決問題的能力

❌ 傳統被動問法

向 AI 索取一段代碼或一個知識定義，複製貼上，但下次遇到類似問題依舊不會解。

💡 費曼法主動出擊

讓 AI 扮演思維教練追問你，迫使你用大白話解釋概念。只有能跟六歲小孩解釋清楚的事物，才是你真正內化的知識。

建立你的學習階梯

讓 AI 為你量身打造你的學習路徑：

對 AI 貼上以下 Prompt
我想一步步學會「使用 AI agent 來幫助我的工作與生活」，請先問我三個問題：
1. 我現在對 AI agent 的了解程度？（1 新手 ~ 5 能獨立操作）
2. 我希望 2 個月後能做到什麼具體的事？
3. 我過去學新東西最大的障礙是什麼？
根據我的回答，幫我建立一個五級學習階梯，每級都要有：里程碑、3 題自檢問題、過了這關後的下一步。

動手做：費曼法極速體驗

🌅 1. 開工對齊

發送 Prompt：開工！今天我要用費曼法練習解釋『AI 助理』。請扮演蘇格拉底式的思維教練，用六歲小孩聽得懂的邏輯向我追問...

🧠 2. 費曼對決

回答教練的挑戰，試著不用專業術語，用你自己的白話把「AI 助理」解釋清楚。

🌌 3. 收工存檔 (SSOT)

發送：收工！請幫我整理今天理解的重點與盲點到我的 SSOT 小卡中，並清理暫存。 複製回覆並安全封存！
