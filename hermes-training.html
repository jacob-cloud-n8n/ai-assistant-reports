PART 3

登入駕駛 — 開始設定你的貼身助理

確定選擇專屬休旅車 Hermes Agent 後，跟著以下步驟發動引擎、上路試駕！

1
建立 Telegram 互動介面 (設定目的地)

我們使用普及率極高、支援跨平台且免費的 Telegram 作為與 Hermes 的互動介面。

2
申請 OpenRouter 統一模型路由 (拿到鑰匙)

OpenRouter 可以讓你免去管理多個 AI 帳號的麻煩，一個 API 就能依需求自由切換 GPT-4o、Claude 或 Gemini 模型。

3
第一次點火啟動 (基本試駕對話)

將 Token 寫入 Hermes 設定中，發動引擎並在 Telegram 進行初次對答測試。

# 本機啟動您的 Hermes 助理
npm install
npm run dev

# 或是透過 CLI 啟動 runtime
hermes start --bot telegram
