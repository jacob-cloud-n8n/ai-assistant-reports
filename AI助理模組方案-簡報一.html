基礎模組 · 必選

🤖 LINE AI 智能回覆

LINE 訊息進來，AI Agent 用你的知識庫自動回覆客戶。
  • ✓ 雙大腦備援機制：Gemini 主腦 + DeepSeek/Groq 備援，99.9% 在線保障
  • ✓ 防插話防重發鎖定：智慧 Session 隊列，防止客戶連傳訊息導致 AI 重複回覆
  • ✓ 雲端動態提示詞載入：動態讀取指示，免改代碼即時調整 AI 語氣與邏輯
  • ✓ 多模態圖片分析：整合 ImgBB 自動上傳，AI 可直接看客戶傳來的圖片（如收納空間）進行估價
實戰案例：美地居家收納、小牧人羊奶 — 客戶 LINE 問問題/傳現場照片，AI 秒回專業評估

🧑 真人轉接

AI 答不了的問題，自動轉接真人客服處理。
  • ✓ 智慧意圖分流工具：AI 自動研判客戶要求真人/報價等意圖，調用轉接工具
  • ✓ 跨平台即時通報：自動觸發子流程，發送 Discord/Telegram 通知老闆
  • ✓ 完整對話脈絡保留：真人接手時可直接查看 AI 與客戶先前的完整聊聊記錄
  • ✓ 上班時間直接轉接，非上班時間引導留單並通知
運作方式：AI 觸發轉接工具 → Discord/Telegram 即時推送通知 → 真人無縫接手

📦 預約/訂單自動化

引導客戶填寫訂單資訊，自動建檔到 Google Sheets，通知老闆。
  • ✓ Google Sheets 雙軌讀寫：AI 助理能直接調用工具讀寫試算表，記錄預約或查詢價格
  • ✓ Notion 預約看板同步：自動於 Notion 建立客戶預約卡片，方便團隊協作
  • ✓ LINE 互動引導（品項、數量、預約時間、地址⋯）
  • ✓ Telegram 訂單即時推送，零遺漏、零人工謄寫
實戰案例：小牧人羊奶 — 客戶填單 → 自動寫入 Sheets 與 Notion → 老闆即時收到通知

📢 訂單狀態推播

出貨、配送、完成，每個狀態自動 LINE 通知客戶。
  • ✓ Notion 狀態看板聯動：在 Notion 看板拖曳訂單狀態，自動觸發 LINE 推播
  • ✓ 配送通知自動化：串接配送系統，自動將物流單號發送至客戶 LINE
  • ✓ 訂單確認、出貨通知、送達確認、售後評價邀請全流程自動化
  • ✓ 客戶不用一直來問「我的商品寄出了嗎？」
實戰案例：小牧人羊奶 — Notion / Google Sheets 狀態更新後，自動發送 LINE 通知

📣 主動推播通知

新品上市、促銷活動、季節提醒，主動推播給客戶。
  • ✓ 排程中樞定時觸發：根據設定時間自動執行推播，如季節提醒（該播種/收納了！）
  • ✓ 客戶分群精準行銷：依據 Google Sheets/Notion 歷史標籤分群發送
  • ✓ 促銷活動、限時折扣、新品上市訊息群發與點擊追蹤
運作方式：排程中樞觸發 → 讀取分群名單 → n8n 自動發送 LINE 群發訊息
